# Nsec Trading Desk

Replaced the geo-blocked Binance WebSocket (451 handshake failures) with short-interval polling of the geo-friendly Binance REST mirrors (data-api.binance.vision / api.binance.us, with proxy fallback). useLivePrices now synthesizes real tick direction, flash, and rolling sparklines from live 24hr ticker data; useCandles polls latest klines for the forming candle. All numbers remain real fetched data. Removed now-unused buildStreamUrl/parseKline/useBinanceSocket usage.
